# ColombiaFinanciera

ColombiaFinanciera es un sitio web que reúne explicaciones simples sobre temas financieros en Colombia.

El objetivo es servir como punto de entrada a la comunidad en Reddit, donde ocurren las discusiones y se comparten experiencias reales:

https://www.reddit.com/r/ColombiaFinanciera

## Enfoque

* No es asesoría financiera
* El contenido es introductorio
* La comunidad es la fuente principal de contexto

## Contenido

Los artículos están basados en temas que aparecen con frecuencia en la comunidad y buscan dar una primera orientación antes de profundizar.

## Desarrollo

```bash
npm install
npm run dev
npm run build
```

## Nota

Evitar contenido promocional o lenguaje de marketing.
El objetivo es claridad, no persuasión.
